# hw-data-types

Создать переменные x = -5, y = 9, z = 6, s = '1cm'.
Расписать в комментарии очередность действий (по приоритету) для каждого выражения. Вывести выражения в консоль. 

 x += y++ / 3;
 
 z /= --x - 2;
 
 z /= y % 2;
 
 y *= z + s;
 
 x = +x + s;
